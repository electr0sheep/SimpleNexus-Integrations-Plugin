## Installation
Clone the repo  
`git clone git@github.com:electr0sheep/SimpleNexus-Integrations-Plugin.git`

Install using apm  
`cd SimpleNexus-Integrations-Plugin`  
`apm link`

## Uninstallation
Uninstall using apm
`cd SimpleNexus-Integrations-Plugin`
`apm unlink`
